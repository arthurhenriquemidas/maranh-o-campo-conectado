# Plataforma Jurídica - Protótipo

Este é um protótipo de uma plataforma jurídica simples, com páginas de login, cadastro de cliente, cadastro de advogado, painel do cliente, painel do advogado, painel do administrador e cadastro de processo.

## Como testar o protótipo

1. **Clone ou baixe este repositório em sua máquina.**
2. **Abra a pasta do projeto no seu computador.**
3. **Clique duas vezes no arquivo `index.html` para abrir no navegador.**

## Como acessar as páginas

- **Página de Login:**
  - É a página inicial ao abrir o `index.html`.
  - Preencha e clique em "Login" (simulação, não há autenticação real).

- **Cadastro de Cliente:**
  - Clique em "Cadastre-se como Cliente" na tela de login.
  - Preencha os dados e clique em "Cadastrar".

- **Cadastro de Advogado:**
  - Clique em "Cadastre-se como Advogado" na tela de login.
  - Preencha os dados e clique em "Cadastrar".

- **Painel do Cliente:**
  - Após simular o login como cliente, você será direcionado ao painel do cliente.
  - Aqui é possível cadastrar novos processos e visualizar os processos cadastrados.

- **Painel do Advogado:**
  - Após simular o login como advogado, você será direcionado ao painel do advogado.
  - Aqui é possível visualizar os processos aceitos.

- **Painel do Administrador:**
  - Após simular o login como administrador, você será direcionado ao painel do administrador.
  - Aqui é possível visualizar processos disponíveis para atribuição.

## Observações

- O protótipo não possui backend, banco de dados ou autenticação real. Todas as ações são simuladas no frontend.
- Para testar diferentes fluxos, utilize os botões de navegação e cadastro.
- O layout é responsivo e pode ser testado em diferentes tamanhos de tela.

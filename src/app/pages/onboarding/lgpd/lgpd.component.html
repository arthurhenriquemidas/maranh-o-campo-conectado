LEI GERAL DE PROTEÇÃO DE DADOS - LGPD

Lei nº 13.709/2018 - Em vigor desde setembro de 2020

A Lei Geral de Proteção de Dados (LGPD) estabelece regras claras sobre como as empresas devem coletar, armazenar, tratar e compartilhar dados pessoais dos cidadãos brasileiros. Na 4Jus, estamos totalmente comprometidos com o cumprimento desta legislação.

1. O que é a LGPD?

• Lei federal que regula o tratamento de dados pessoais no Brasil

• Aplicável a qualquer pessoa ou empresa que trate dados pessoais

• Inspirada no Regulamento Geral de Proteção de Dados (GDPR) europeu

• Entrou em vigor em setembro de 2020

• Fiscalizada pela Autoridade Nacional de Proteção de Dados (ANPD)

2. Princípios fundamentais da LGPD:

• Finalidade: Dados devem ser usados apenas para objetivos legítimos

• Adequação: Tratamento compatível com as finalidades informadas

• Necessidade: Limitação ao mínimo necessário para cumprir objetivos

• Livre acesso: Garantia de consulta gratuita sobre dados tratados

• Qualidade: Dados devem ser exatos, claros e atualizados

• Transparência: Informações claras sobre tratamento de dados

• Segurança: Medidas técnicas e administrativas de proteção

• Prevenção: Medidas para prevenir danos decorrentes do tratamento

• Não discriminação: Não usar dados para fins discriminatórios

• Responsabilização: Demonstração de medidas efetivas de proteção

3. Dados pessoais vs. Dados sensíveis:

• Dados pessoais: Qualquer informação que identifique uma pessoa

• Dados sensíveis: Informações que podem gerar discriminação

Dados pessoais incluem:

• Nome, CPF, RG, endereço, telefone, e-mail

• Dados de localização, IP, cookies

• Informações profissionais e acadêmicas

Dados sensíveis incluem:

• Origem racial ou étnica

• Convicções religiosas ou filosóficas

• Opiniões políticas, filiação sindical

• Dados genéticos ou biométricos

• Dados de saúde

• Orientação sexual

4. Bases legais para tratamento de dados:

• Consentimento: Autorização expressa e informada do titular

• Legítimo interesse: Quando necessário para atividades legítimas

• Cumprimento de obrigação legal: Quando exigido por lei

• Execução de contrato: Para cumprir obrigações contratuais

• Proteção do crédito: Análise de risco em operações financeiras

• Exercício regular de direitos: Em processos judiciais ou administrativos

• Proteção da vida: Quando necessário para salvar vidas

• Tutela da saúde: Em procedimentos de saúde

• Interesse público: Para políticas públicas

• Estudos e pesquisas: Com anonimização dos dados

5. Seus direitos como titular de dados:

• Confirmação: Confirmar se tratamos seus dados pessoais

• Acesso: Acessar seus dados a qualquer momento

• Correção: Corrigir dados incompletos ou inexatos

• Eliminação: Solicitar exclusão de dados desnecessários

• Portabilidade: Receber dados em formato estruturado

• Oposição: Opor-se ao tratamento baseado em legítimo interesse

• Revogação: Retirar consentimento a qualquer momento

• Explicação: Receber informações sobre decisões automatizadas
